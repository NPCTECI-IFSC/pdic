PDIC - Rotas Estratégicas Setoriais

Com o Projeto Rotas Estratégicas Setoriais, o Sistema FIESC tem o objetivo de sinalizar caminhos de construção do futuro para cada um dos setores e áreas identificados no Projeto Setores Portadores de Futuro para o Estado de Santa Catarina. Os setores selecionados são considerados como os mais promissores para a indústria do estado no horizonte de dez anos, até 2022. A partir dessa identificação, evidencia-se a necessidade de concepção de mapas de trajetórias a serem percorridas para ampliar a competitividade de cada um dos setores.
